# Voynich写本 スロット文法 V8

## 概要

V8文法は、Voynich写本に出現する全8078ユニーク語を**完全に生成できる**スロット文法である。
ベースとなるスロット構造（V4文法）を最大4回連結することで全語彙をカバーする。

---

## スロット構造

各スロットは「その位置で使用できる文字列の候補」を表す。
単語生成時は、スロット0から15を順に参照し、各スロットで候補のいずれか一つを選択するか、またはそのスロットをスキップする（最低1スロットは必ず使用）。
複数候補が同一位置にマッチする場合は**スロット番号の小さい方**、同一スロット内では**リストの前の候補**を優先する（貪欲マッチ）。

| スロット | 候補 | 備考 |
|----------|------|------|
| Slot 0 | `l` `r` `o` `y` `s` `v` | 語頭要素（sonorant・母音・希少字） |
| Slot 1 | `q` `s` `d` `x` `l` `r` `h` `z` | 語頭子音群（希少字を含む） |
| Slot 2 | `o` `y` | 短母音 |
| Slot 3 | `d` `r` | 子音 |
| Slot 4 | `t` `k` `p` `f` | 無声阻害音 |
| Slot 5 | `ch` `sh` | 二字綴り |
| Slot 6 | `cth` `ckh` `cph` `cfh` | 三字綴り |
| Slot 7 | `eee` `ee` `e` `g` | e系列 |
| Slot 8 | `k` `t` `p` `f` `ch` `sh` `l` `r` `o` `y` | 中間・後続子音 |
| Slot 9 | `s` `d` `c` | 子音 |
| Slot 10 | `o` `a` `y` | 母音 |
| Slot 11 | `iii` `ii` `i` | i系列 |
| Slot 12 | `d` `l` `r` `m` `n` | 子音（語末寄り） |
| Slot 13 | `s` | s |
| Slot 14 | `y` | y |
| Slot 15 | `k` `t` `p` `f` `l` `r` `o` `y` | 語末要素 |

---

## 生成規則

### 基本単語（ベース）

スロット0〜15を順に適用して生成される文字列1つを**ベース**と呼ぶ。

```
ベース ::= スロット0〜15の順次適用（最低1スロット使用）
```

### V8単語

**ベースを1〜4個連結**したものがV8単語である。

```
V8単語 ::= ベース
         | ベース + ベース
         | ベース + ベース + ベース
         | ベース + ベース + ベース + ベース
```

---

## V4からV8への変遷

| バージョン | 変更内容 | マッチ率 |
|-----------|---------|---------|
| V4 | 基本スロット文法（ベース×1） | — |
| V5 | ベースの繰り返し上限を3回に拡張 | — |
| V6 | 繰り返し上限を4回に拡張 | ~99.9% |
| V7 | Slot 0 に `v` を追加（f57v 限定グリフ） | 99.975% |
| V8 | Slot 1 に `z` を追加（`tazain`・`zepchy` 対応） | **100.0%** |

---

## V8で追加された要素

### `v`（Slot 0）

- 出現箇所: **f57vのみ**
- 出現語: `v`, `vo`, `vs`（計3語）
- 他グリフとの混同可能性なし
- 語頭専用の振る舞いのため Slot 0 に配置

### `z`（Slot 1）

- 出現語: `zepchy`, `tazain`（計2語）
- 他グリフとの混同可能性なし
- Slot 1 の他メンバー（`q`, `x`）と同様に語頭子音として振る舞う
- `tazain` は `ta` + `zain` の2ベース連結として解析される

## 生成例

| 単語 | 解析（ベース分割） | スロット列 |
|------|-----------------|-----------|
| `qokeedy` | `qokeedy`（単体） | s1(q)+s2(o)+s8(k)+s7(ee)+s12(d)+s14(y) |
| `chedy` | `chedy`（単体） | s5(ch)+s7(e)+s12(d)+s14(y) |
| `tazain` | `ta` + `zain` | s4(t)+s10(a) ／ s1(z)+s10(a)+s11(i)+s12(n) |
| `zepchy` | `ze` + `pchy` | s1(z)+s7(e) ／ s4(p)+s5(ch)+s14(y) |
| `v` | `v`（単体） | s0(v) |
