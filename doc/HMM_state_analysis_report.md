# ヴォイニッチ手稿 HMM 状態別デコード分析レポート

**分析日**: 2026-03-03
**分析スクリプト**: `hypothesis/01_bigram/source/slot_hmm_state_analysis.py`
**対象**: ヴォイニッチ手稿テキスト (`data/voynich.db` — `words_enriched` テーブル)

---

## 1. 概要

本レポートは、ヴォイニッチ手稿の単語列に対して **隠れマルコフモデル (HMM)** を適用し、各隠れ状態が単語内のどの文字クラスを、どの位置（語頭・語中・語末）で担っているかを定量的に分析した結果をまとめるものである。

状態数 **k=7** および **k=8** の Full HMM を対象に、全ユニーク単語（長さ ≥ 2）を Viterbi デコードし、状態ごとの占有率・位置内訳を集計した。

---

## 2. 分析手法

### 2.1 HMM の構造

| パラメータ | 値 |
|---|---|
| モデル種別 | Full HMM（遷移方向の制約なし） |
| 状態数 | k = 7, 8 |
| 語彙 | ヴォイニッチ文字 + BOS(`^`) / EOS(`$`) / PAD(`_`) |
| 学習試行数 | 20 回のランダム再スタート |
| 最大反復回数 | 200 回（収束判定 tol=1e-4） |
| デバイス | GPU (CUDA) / CPU フォールバック |

### 2.2 学習・デコード手順

1. **前処理**: 各単語に BOS・EOS トークンを付加し、バッチ行列へパディング
2. **学習**: Baum-Welch アルゴリズム（前向き・後ろ向き法）を PyTorch でバッチ並列化
3. **最良モデル選択**: 20 試行中で対数尤度が最大の初期化を採用し `.npz` にキャッシュ
4. **デコード**: Viterbi アルゴリズムで各単語に状態パスを割り当て
5. **位置分析**:
   - BOS・EOS を除いた単語本体のみを対象
   - **語頭**: 位置 0
   - **語中**: 位置 1 〜 L-2（L < 3 の場合は N/A）
   - **語末**: 位置 L-1

### 2.3 モデル選択の根拠（BIC/AIC）

前段階の実験（`hmm_pt_max` ディレクトリ）で k=2〜8 を比較した結果:

| k | 対数尤度 (Full) | BIC |
|---|---|---|
| 2 | −162,021 | 324,742 |
| 3 | −149,784 | 300,657 |
| 4 | −143,440 | 288,381 |
| 5 | −137,349 | 276,634 |
| 6 | −131,585 | 265,561 |
| **7** | **−126,899** | **256,668** |
| **8** | **−122,904** | **249,178** |

BIC は単調減少しており、k=7・8 が最も高い表現力を持つ。k=7・8 を対象に状態解釈を行った。

> **注**: Full HMM は Left-to-Right HMM より常に高い対数尤度を達成する。
> k=7 では Full=−126,899 対 L-to-R=−134,963 と大差があり、ヴォイニッチ単語の文字遷移が**一方向の連鎖モデルに収まらない**ことを示唆する。

## 3. k=7 の状態解釈

### 3.1 各状態の特徴サマリー

| 状態 | 主要文字クラス | 典型語例 | 位置傾向 | 命名案 |
|---|---|---|---|---|
| S0 | `e`, `ee`, `eke` | ek, eeedy, eteedy | 語頭・語中優位、語末回避 | **E-cluster（e連続）** |
| S1 | `c`, `ch`, `sh` | ch, co, cs, cy, sh | 語頭優位 | **C-onset（ch/sh 語頭子音）** |
| S2 | `a`, `l`, `r`, `y` | aiy, ary, aly, alory | 語頭・語末両方 | **A-frame（a語頭フレーム）** |
| S3 | *(占有率 0%)* | *(全単語 0%)* | 未使用 | **Phantom（縮退状態）** |
| S4 | `o`, `q`, `l` | dytoly, qokaoy, otoly | 語頭・語中優位、語末回避 | **O/Q-cluster（o/q 中核音）** |
| S5 | `i`, `n`, `r`, `l` | iiin, iin, in, rn | 全位置（語頭〜語末）均等 | **I-run（i・流音連続）** |
| S6 | `h`, `hh`, `cth` | cth, ckhhy, chckhh | 語中・語末優位、語頭回避 | **H-coda（h後続コーダ）** |

### 3.2 状態別詳細

#### S0: E-cluster（e 連続体）

- **占有率上位語**: `ek` (100%), `deeeese` (85.7%), `eeedeed` (85.7%), `eeody` (80%)
- **位置内訳**: 語頭 100% / 語中 83〜100% / **語末 0〜100%**（語末は限定的）
- `ee`, `eee` など e の繰り返しで構成される単語を ほぼ独占的に担う
- ヴォイニッチにおいて e 系文字は最高頻度文字の一つであり、専用状態の形成は妥当

#### S1: C-onset（ch/sh 語頭子音クラスター）

- **占有率上位語**: `ch`, `co`, `cs`, `cy`, `sh` (各 50%), `chchsty` (42.9%)
- **位置内訳**: **語頭 100%** がほぼ全語で確認、語中はやや低め、語末は稀
- `ch`・`sh` はヴォイニッチで最頻出の文字ペアであり、語頭に集中する性質が再現された
- 占有率の上限が 50% 程度にとどまるのは、`ch`・`sh` が 2 文字語では必ず半分を占めるためであり、自然な上限

#### S2: A-frame（a 語頭フレーム）

- **占有率上位語**: `aiy`, `aly`, `ary`, `amy` など(各 66.7%), `aloly` (60%)
- **位置内訳**: **語頭と語末の両方で高スコア**（各 66〜100%）、語中は低い
- 「語頭の `a`〜語末の `y/r/g`」というフレーム構造の単語を担う
- これは接頭辞 `a-` + 語幹 + 接尾辞 `-y` の形態論的パターンを示唆する可能性がある

#### S3: Phantom（縮退状態）

- **全 30 語の占有率が 0%**
- 「占有率上位 30 語」でさえ 0% という事実は、この状態が**実質的に使用されていない**ことを意味する
- 20 回ランダム再スタートの最良モデルでも残存する縮退状態であり、モデルが k=7 に対して若干過剰パラメータである可能性を示す
- k=8 の S4 でも同様の縮退状態が出現（再現性あり）

#### S4: O/Q-cluster（o/q 中核音）

- **占有率上位語**: `dytoly` (83.3%), `qokaoy` (83.3%), `otoly` (80%), `ololy` (80%)
- **位置内訳**: 語頭・語中で強く、**語末は 0%**
- `q` は常に `o` の前に出現するというヴォイニッチの著名な制約（qo 語頭拘束）を部分的に担う
- `o`, `l`, `k`, `t` など中核的な「丸い」文字を含む単語に集中

#### S5: I-run（i・流音連続）

- **占有率上位語**: `iiin` (100%), `iin` (100%), `in` (100%), `lr` (100%), `rn` (100%)
- **位置内訳**: **語頭・語中・語末すべて高スコア**（単語全体を支配）
- `i` の連続（`ii`, `iii`, `iiii`）および流音・鼻音 (`r`, `l`, `n`) が独立したクラスを形成
- ヴォイニッチにおける `aiin`, `aiiin` 型接尾辞との対応が注目される
- 単語全体を占有する傾向は「この状態の文字だけで成立する語形」の存在を示す

#### S6: H-coda（h 後続コーダ）

- **占有率上位語**: `cth` (66.7%), `shckhh` (66.7%), `chckhhhy` (62.5%)
- **位置内訳**: 語頭 **0%**（完全に語頭回避）、語中・語末で出現
- `h`, `hh`, `khh`, `phh` などの `h` 後続クラスターを語中〜語末に担う
- ヴォイニッチの `ch`, `sh` が語頭に偏るのに対し、孤立 `h`・`hh` は内部/末尾位置に現れるという非対称性を反映

## 4. k=8 の状態解釈

### 4.1 各状態の特徴サマリー

| 状態 | 主要文字クラス | 典型語例 | 位置傾向 | 命名案 |
|---|---|---|---|---|
| S0 | `q`, `o`, `l` | qol, qoor, olqo, yqor | 語頭・語中優位 | **Q-onset（q語頭クラスター）** |
| S1 | `a`, `d`, `y` | aiy, amy, day, dady | 語頭・語末両方 | **A/D-frame（a/d 交替フレーム）** |
| S2 | `c`, `h`, `sh` | sh, cs, shhy, chhy | 語中・語末 | **H-medial（h 語中クラスター）** |
| S3 | `e`, `ee` | eeeey, eeey, eeer | 語頭・語中優位 | **E-cluster（e連続）** |
| S4 | *(占有率 0%)* | *(全単語 0%)* | 未使用 | **Phantom（縮退状態）** |
| S5 | `d`, `k`, `t` | ded, ked, ted, da, dady | 語頭優位 | **D-onset（d/k/t 語頭子音）** |
| S6 | `c`, `ch`, `ck` | scth, cth, chckh | **語頭**優位 | **C-onset（ch 語頭クラスター）** |
| S7 | `i`, `n`, `r` | iiin, iin, in, rn | 全位置均等 | **I-run（i・流音連続）** |

### 4.2 k=7 との対応と相違点

k=8 では k=7 の状態が分化・精細化された:

| k=7 状態 | k=8 対応状態 | 変化 |
|---|---|---|
| S0 (E-cluster) | S3 (E-cluster) | ほぼ同一 |
| S1 (C-onset) | S6 (C-onset) + S2 (H-medial) | **語頭 `ch` と語中 `h` が分離** |
| S2 (A-frame) | S1 (A/D-frame) | `d` 語頭語を取り込み拡張 |
| S4 (O/Q-cluster) | S0 (Q-onset) | `q` 語頭に特化 |
| S5 (I-run) | S7 (I-run) | ほぼ同一 |
| S6 (H-coda) | S2 (H-medial) | 語中特化に精細化 |
| *(新規)* | S5 (D-onset) | **`d`/`k`/`t` 語頭子音が独立分離** |

最も注目すべき変化は **ch/sh の分化**:
- k=7: S1 が「ch/sh 全般」を担っていた
- k=8: S6（語頭の `ch`）と S2（語中の `h` クラスター）に分離

これは、**同一文字列 `ch` が語内位置によって異なる機能を持つ**という仮説を支持する。

## 5. 主要知見（洞察）

### 知見 1: ヴォイニッチ語の文字は少なくとも 5〜6 の機能的クラスに分類できる

k=7・8 の両モデルで**再現性をもって**出現するクラスターは以下の通り:

| クラス | 代表文字 | 機能的役割（仮説） |
|---|---|---|
| E-cluster | `e`, `ee`, `eee` | 母音連続・コア音節核 |
| C/CH-onset | `c`, `ch`, `sh` | 語頭子音クラスター |
| A-frame | `a`, `al`, `ar`, `y` | 語頭/語末フレーム（接辞？） |
| O/Q-cluster | `o`, `q`, `l`, `k` | 中核音節（円唇？） |
| I-run | `i`, `ii`, `iii`, `n`, `r` | 鼻音・流音連続（接尾辞？） |
| H-medial | `h`, `hh`, `kh`, `ph` | 語中コーダ子音 |

### 知見 2: 縮退状態（Phantom State）の再現的出現

k=7 の S3 および k=8 の S4 で**全単語の占有率が 0%**という縮退状態が確認された。これは：

- モデルが学習データを表現するのに k=7〜8 の状態数が若干過剰であることを示す
- または、この状態が「BOS/EOS 位置専用」の遷移ハブとして機能している可能性がある
- 実際の使用上有効な状態数は k=6〜7 程度かもしれない

### 知見 3: Full HMM が Left-to-Right HMM を大幅に上回る

k=7 での差: Full −126,899 vs L-to-R −134,963（差 = 8,064 nats）

これはヴォイニッチ語の文字遷移が**単純な左から右への順序制約に収まらない**ことを示す。具体的には:
- 語末近くの状態から語頭近くの状態へ戻る遷移（ループ）が実際に存在する
- 繰り返し構造（`ee`, `okeoke` など）が双方向遷移を必要とする
- または、長距離依存（語頭の文字が語末を制約する）が存在する

### 知見 4: 「i-run」状態の全位置占有は形態論的接尾辞を示唆する

`iiin`, `iin`, `in` のような語が S5/S7 を 100% 占有し、かつ語頭・語中・語末のすべてで高スコアを示す。これは：

- `aiin`, `aiiin`, `oriin` など「a/o + i連続 + n/r」型の語末パターンが独立した形態素（接尾辞）として機能している可能性
- ヴォイニッチ研究で既に注目されている `-aiin` 型接尾辞との一致
- この接尾辞が語末のみならず一部の語では語全体を支配する（語根なしで接尾辞のみの語）

### 知見 5: 語頭と語中の ch/sh は機能が異なる

k=8 での分化（S6: 語頭 ch vs S2: 語中 h）は、同一の書記素 `ch`・`sh` が：

- **語頭**: 独立した音節頭子音クラスターとして機能
- **語中**: コーダまたは介入子音として異なる役割を担う

という形態音韻論的二重機能の可能性を示唆する。これは表音文字仮説において重要な示唆であり、`ch` が単一の音素ではなく文脈依存の音素連鎖を表す可能性がある。

### 知見 6: q-o 連鎖の状態への反映

k=8 の S0 では `qol`, `qoor`, `qos`, `qot` など q 語頭語が集中。これはヴォイニッチの有名な制約「q の直後には必ず o が来る」を HMM が自律的に学習・独立状態として分離したことを意味する。この q-o の強固な連鎖は：

- `q` が文字ではなく q+o の複合グリフである可能性
- または `q` が語頭マーカー（冠詞・前置詞的要素）として機能する可能性
を示唆する。

## 8. 結論

ヴォイニッチ手稿の単語を HMM (k=7, k=8) で分析した結果、文字列は以下の **5 つの機能的クラスター** に収束することが確認された:

1. **E-cluster**: `e` 系の母音・二重音字連続（最頻出単語群を支配）
2. **CH-onset**: `ch`/`sh` 語頭クラスター（語頭専用の子音機能）
3. **I-run**: `i`/`ii`/`in` 鼻音・流音連続（形態論的接尾辞の候補）
4. **O/Q-core**: `o`/`q`/`l` 系の語核音節（q-o 制約を内包）
5. **H-medial**: `h`/`hh` 系語中コーダ（語頭位置とは別機能）

これらのクラスターは k を変えても再現的に出現しており、ヴォイニッチ手稿の単語構造に**統計的に実在する音韻・形態論的な分節**が存在することを強く示唆する。ランダムな暗号文や無意味文字列であれば、こうした位置特化型の機能分化は出現しにくい。
