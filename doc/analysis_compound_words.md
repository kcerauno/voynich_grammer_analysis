### 考察

- 全語彙の**約4割が複合語**という解釈になる
- 4基複合語は5語のみで、ほとんどは2基
- `chetesescher` → `chet + es + es + cher` のように、**同一ベースの繰り返し**も確認できる
- 「複合語」解釈は純粋な「繰り返し」解釈と数学的に等価だが、`等しい語基が反復している`、`接頭辞的な短ベース（a, ad, aiin など）が多数ある`といった特徴が見えてくる

## 洞察

### 1. 二項結合が支配的なコアルール

```
1基 : 2基 : 3基 : 4基
58% : 39% : 2.7% : 0.06%
```

1→2への変化（倍率0.67）に比べ、2→3への変化（倍率0.07）は**一桁落ちる**。
これは「3基以上の形成を禁じるような強い制約」が存在することを示唆する。
つまり文法の実質的なコアは「**V4基語 ＋ 任意の1回だけの複合**」と見なせる。

---

### 2. 言語学的普遍性との一致

自然言語の複合語でも、**二項分岐（binary branching）**が強く優先される（英語 `bookshelf = book + shelf`、日本語の二字熟語など）。認知的・処理的コストが複合の深さに対して非線形に増大するためである。
ボイニッチ写本がこれと同じ分布パターンを示すことは、**この文書が自然言語の統計的特性を持つ**、あるいは人間の産出規則に沿って生成されたという状況証拠になる。

---

### 3. 形態論的解釈の二候補

| 解釈 | 2基語の見方 | 根拠 |
|------|-----------|------|
| **複合語** | 語根 + 語根 | 2基語が多数、単体語に近い量 |
| **屈折・派生形** | 語根 + 接辞 | 短いベース（`a`, `ad`, `s`, `y`）が頻出 |

どちらでも「**単体語と2基語で語彙の97%を生成できる最小の体系**」であることは変わらず、余分な複雑さを持たない経済的な文法である。

---

### 4. 3基以上は「特殊カテゴリ」の可能性

3%という比率と、4基語がわずか5語という事実から、3基以上の語は：

- **固有名詞・専門語**（医学草本・天文章の特定葉に偏在する可能性）
- **筆写エラー**（スペース抜け、語境界の誤認）
- **稀な文法形式**（複雑格・接続形など）

のいずれかと考えられる。この5語（`chetesescher`, `chkaidararal` など）の**出現葉・文脈を確認することが次の検証ステップ**として有効である。

---

### 5. 文法の「完全性」に関する示唆

V4の繰り返し上限を4に設定することで100%カバーできたが、実質的には：

```
  1基（58%） + 2基（39%） = 97%  ← 実効文法
  3基（2.7%）+ 4基（0.06%） = 3%  ← 周縁・例外
```

これは**V4文法そのものが「中核的な語形成規則」であり、複合は一段階まで**という仮説を強く支持する。V8の「最大4回」は理論的上界に過ぎず、**V5（最大2基）で語彙の97%を説明できる**ことを意味する。
